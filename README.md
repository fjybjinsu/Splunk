# 통합 관제 프로그램
## 1. 개요
- splunk를 사용하는 군 내부 인트라넷이 2개 있는데 각 네트워크에서 사용했던 SPL문

## 2. 스택
- Splunk SPL

## 3. 제작 동기
- 로그 통합 분석
- Threat Hunting 

## 4. 역할
- 양진수 :  대시보드 제작, SPL 작성

## 5. 배운 것
- spl 구문
- splunk 대시보드 제작
- 예상되는 시나리오

## 6. 아쉬운 점
- 공식문서, 상급부대 자료, 인터넷에서 검색하여 작성하였는데 조금 더 다양한 시나리오에 대한 학습이 있었으면 좋겠다.
- 아무래도 빅데이터 기반이다 보니, 운영하면서 측정되는 데이터를 반영하는 것이 조금 더 적극적으로 시도 되었어야 함
